API Fetch - Requisições e respostas vindas do servidor do lado do backend.

a função fetch facilita bastante trabalhar com requisições e manupular as respostas das requisições.

É muito utilizado no react e em outras frameworks de java script.

Portanto, ao ver utilizado o fetch emalgum projeto, saiba que é para manipular as requisições.

Desvantagem ao utilizar o fetch
Suporte para navegadores antigos.
